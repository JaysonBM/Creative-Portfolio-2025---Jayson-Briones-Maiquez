Unity Charity Website Redesign

This project involves the redesign of Unity Charity's E-commerce website, focusing on typography and layout enhancements to provide users with a more engaging experience and a seamless checkout process.

Role

  • Researcher
  • UX Designer
  • UI Designer
  • Web Designer
  • Web Developer

Project Type

  • Web Design & Developing

Tools Used

  • Adobe XD
  • Adobe Illustrator
  • Adobe Photoshop
  • Visual Studio Code
  • Contentful

Design Team 1

  • Jayson Briones-Maiquez

Duration

  • January 2023 - April 2023

Problem + Solution

The original website's layout and typography raised concerns about user experience and readability. Key questions included: Would users navigate the site smoothly? Were there any usability issues? Was the typography clear and legible?

To address these concerns, I focused on organizing sections for clarity and user engagement. Additionally, I enhanced the website's visual appeal to improve user interaction.

Methodology

Research into the original website's design guided the project, followed by the creation of user personas to understand the target audience better. Sketches and wireframes in Adobe XD visualized the layout, leading to a prototype. The website was then developed using Visual Studio Code and launched on Netlify.

Implementations

Improvements included replacing the desktop Hamburger menu with a clear navigation system and enhancing UI buttons for a more cohesive brand experience.

Results + Impact

The project yielded positive outcomes, though opportunities for further learning were identified. Continued optimism and a commitment to growth underscored the project's success.

Visitors now enjoy an enhanced website experience, navigating with ease and without frustration.

Lessons Learned

Key takeaways included refining layout design, researching competitive e-commerce layouts, and ensuring responsiveness across devices, contributing to the improvement of design systems.

While this project served as a valuable learning experience, there is potential for future enhancements. Reflecting on this project will inform future endeavors.

Conclusion

In conclusion, this website redesign project expanded my skills as a UX Designer, encompassing UI Design and Web Development. It was both challenging and rewarding, and I hope to present it to Unity Charity to showcase my capabilities and potentially contribute further to their digital presence.
